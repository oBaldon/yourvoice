# ADR-0002 — Host auto-hosteado (sem servidor central obrigatório)

**Status:** Aceito

## Decisão
Um amigo do grupo hosteia **UI + sinalização + SFU** localmente.

## Contexto
O projeto deve funcionar “como servidor de jogo”: o host sobe localmente e compartilha um endereço (Direct/VPN).

## Motivação
Preserva privacidade, reduz custo e evita dependência de infra pública para o modo Free.

## Consequências
- Disponibilidade depende do host.
- Desempenho depende do upload do host.
